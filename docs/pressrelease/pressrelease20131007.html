Fringe81、アドテク・ハッカソン『Facebook APAC Ads API Hackathon 2013』に参加
〜日本からは一社のみの選出　シンガポールで開催〜
（＊注）ハッカソン（hackathon）＝「ハック」と「マラソン」の造語。開発者向けイベントとして、通常特定の目的に沿って24時間でサービスを作り上げ、プレゼンテーションを行う。Facebookのいいね！（Like! ）ボタンやタイムラインはハッカソンで開発された。
■ Wikipedia ハッカソン http://ja.wikipedia.org/wiki/ハッカソン#cite_note-25


PMDは、Facebook社がFacebookのプラットフォーム上で、企業や開発パートナーをつなげるためのFacebook社認定プログラムです。

今回、アジア太平洋地域での取組みを拡大するためにハッカソンが企画され、アジア全域から選出された 10 社がアドテクノロジー技術を競います。このうち日本から招待されたのは Fringe81 ただ一社のみとなります。

Fringe81は、Facebook APAC Ads API Hackathonで全力を尽くして戦うとともに、アドテクノロジー技術の向上のため、今後も国内外で多様なプロジェクトに参加して参ります。
